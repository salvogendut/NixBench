# Fantasy artwork

The unicorn window frame and enchanted dock are original, AI-assisted raster
assets produced for NixBench from project-owned visual direction. They are
static presentation layers; window controls, launchers, minimized-window
buttons, movement, and input policy remain native NixBench behavior.

`woods.png` is the project-provided Fantasy wallpaper selected for this theme.
The live renderer reads these fixed PNG files through the validated theme
directory and injects bounded `data:` image URLs into its network-disabled
atlas document.
